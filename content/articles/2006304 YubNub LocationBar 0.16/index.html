--- title: YubNub LocationBar 0.16 date: 2006-10-31 19:20:00 Z tags: ["Tech","Firefox","Yubnub"] wordpress_path: /2006/10/yubnub-locationbar-016 --- Version 0.16 of my YubNub LocationBar Firefox Extension has been released. There is now support for non-standard URLs (such as those from nic.d. Arrow-key history navigation broke some default Firefox features, so it is now Ctrl+Arrow Key. A hackish integration with Firefox auto-complete history has also been coded. Type y: before a command to have the auto-complete work (or just to force it into YubNub mode).
